## Suggested First Release

Suggested tag:

- `v0.1.0`

Suggested release title:

- `TXT Splitter v0.1.0`

Suggested release description:

```md
TXT Splitter v0.1.0 is the first public release of a local desktop tool for splitting long-form text and document files into manageable chunks.

Highlights:

- Intelligent chapter-based splitting with hierarchy detection
- Physical splitting modes by size, words, lines, and paragraphs
- Native support for `.txt`, `.md`, `.epub`, `.docx`, `.mobi`, `.azw3`, and partial `.pdf`
- Dark/Light desktop GUI built with CustomTkinter
- Offline-first local processing for private documents
- Automated regression coverage for core parsing, format loading, Markdown heading support, and GUI smoke tests

Notes:

- PDF support currently works for files with extractable text layers
- OCR is not included in this release
- Windows is the primary supported platform for packaged binaries
```
